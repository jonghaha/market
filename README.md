# market

[ 개발 필수요건 ]
1. 상품관리
- 상품을 등록 / 수정 / 삭제 할 수 있어야 한다.

2. 주문관리
- 등록한 상품으로 주문을 받을 수 있어야 한다.
- 주문한건당 여러개의 상품을 받을 수 있다.(1:N)
- 주문은 부분취소가 가능하다.
- 주문 내 일부 상품만 취소할 수 있다.
- 주문상태 예시 : 주문접수/배송완료/주문취소
- 주문 및 취소 시 알림
- 콘솔에 로그 출력을 알림발송으로 인정합니다.
